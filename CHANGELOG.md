# 更新履歴 / Changelog

このファイルでは、リリースごとの主な機能追加・改善・不具合修正を簡単にまとめます。
細かな内部修正や調整はすべて列挙せず、ユーザーにとって意味のある変更を中心に記録します。

This file summarizes notable changes for each release. It focuses on user-visible features, improvements, and fixes rather than listing every internal change.

## Unreleased

## v1.1.3 - 2026-06-08

### 改善 / Improved

- NovelAIプロンプト分解モードで、`||...|...||` のランダム選択構文と `1.2::tagA, tagB::` などの数値重み指定構文を1つのタグとして扱うように変更。
- NovelAIプロンプト分解モードで、単独の `::` を未閉じの強調/抑制括弧を閉じる区切りとして扱うように変更。
- プロンプト分解モードで無効化したタグを、通常モードへ切り替えてもRAIV専用マーカーで保持し、NovelAI送信時には除外するように変更。
- NovelAI生成画像のファイル名設定で、従来の時刻形式を「日付_時刻.png」として明示し、新たに時刻のみの形式を追加。
- 英語UIのプロンプト分解モードで、強調/抑制ボタンの文字が見切れないよう表示を調整。

### 修正 / Fixed

- アップデート確認で版種に合わないZIP URLを案内しないよう、GitHub Releases一覧への案内に変更。

## v1.1.2 - 2026-06-08

### 改善 / Improved

- エンジン設定の tile を、0〜16 のスピンボックスからプリセット付きの任意入力へ変更。内蔵GPUなどでメモリ不足になる場合に 128 や 256 などを指定しやすくしました。

## v1.1.1 - 2026-06-08

### 追加 / Added

- その他タブに、画像削除時の確認メッセージを表示するオプションを追加。デフォルトはオン。
- エンジン設定に、画像ごとに縦サイズ閾値へ届く最小倍率を自動選択するオプションを追加。

### 改善 / Improved

- NovelAI生成タブ名からβ表記を削除。
- NovelAI生成タブのプロンプト / 除外したい要素欄を、それぞれ個別に上下ドラッグで高さ調整できるように変更。調整した高さは設定へ保存されます。
- NovelAI生成タブのプロンプト分解モードで、タグ一覧の選択フォーカス表示が残らないように調整。
- NovelAI生成ボタンの生成中状態が分かるように、表記と活性状態を調整。
- 倍率フォルダ表示キャッシュが、現在のエンジン、モデル、倍率に完全一致するフォルダだけを使うことをUI説明に明記。
- 自動倍率設定時の画像高さ取得を、可能な限り画像ヘッダ読み取りと専用キャッシュで行うようにし、表示切替や先読み中に元画像を繰り返し読み直してレスポンスが悪化する問題を軽減。

### 修正 / Fixed

- ホイールクリックでボーダーレス全画面へ切り替えて戻った時、ウィンドウサイズが少し小さくなることがある問題を軽減。
- 自動倍率設定時、画像削除後に拡大済み画像キャッシュを取り逃がすことがある問題を修正。

## v1.1.0 - 2026-06-07

### 追加 / Added

- NovelAI生成タブを追加。永続APIトークンを使ったテキストから画像生成、トークンのDPAPI暗号化保存、NovelAI公式表現に寄せた生成設定、アニメ/ケモノモード切替、タグ単位のプロンプト編集、タグプリセット保存/読込/削除、シード値/時刻によるファイル名選択、折りたたみ可能な詳細設定、品質タグ/除外プリセット、日付別保存、Enter生成、推定Anlas表示、生成後の自動表示、拡大処理キュー投入、生成設定JSONサイドカー保存、NovelAI生成画像のメタデータインポートに対応。
- 設定タブの表示/非表示、右ペインの左右移動、Deleteキーによる画像削除と拡大結果の同時削除設定を追加。

## v1.0.0 - 2026-06-05

### 追加 / Added

- PythonをPCへインストールしなくても `RAIV.exe` から起動できるスタンドアロン版ZIPを配布形式に追加
- 従来の軽量なpyw版ZIPと、Python不要のスタンドアロン版ZIPを別ファイルとして配布する方針に変更

### 改善 / Improved

- 画像調整タブの明るさ、コントラスト、ガンマ、シャープネスをスライダー操作へ変更し、各項目に個別リセットを追加。操作中の重さを抑えるため、画像への反映は短い遅延後にまとめて行うように改善
- キーボードショートカットの `R` / `L` を1度ずつの画像回転に変更し、`Shift` + `R` / `Shift` + `L` を90度回転として追加。1度回転はキー長押しで微調整しやすいよう、描画時の回転を即時反映しつつ高品質リサンプルは遅延するように改善
- 回転時はズーム倍率を維持する挙動をデフォルトに変更し、従来のように回転後の画像全体が収まるようフィットし直す挙動は全般タブの設定で切り替え可能にした

### 修正 / Fixed

- Windows の高DPIスケーリング環境で、右ダブルクリックなどの等倍表示が実ピクセル基準の100%にならず、引き延ばされて見える問題を修正
- Windows でボーダーレス全画面表示へ切り替える時、通常ウインドウの再表示を挟まずに枠とサイズを直接変更し、背面ウインドウが一瞬見えやすい問題を軽減。高DPI環境ではWin32の物理モニター座標を使い、左上詰めの小さいウインドウになる問題も修正
- 起動時に右ペイン固定モードの表示状態が復元と噛み合わず、固定中なのに右ペインが非表示のままになる場合がある問題を抑制
- 拡大処理対象外などで比較先画像がない場合も、比較モードでは左右に同じ画像を表示して境界線をドラッグできるように修正
- 画像全体が内接フィット表示されている時は、ドラッグしてもパンしないように修正
- 右ペインの最大幅を画面幅の約50%から約60%へ拡大

## v0.1.5 - 2026-06-02

### 追加 / Added

- 画像調整タブに、画像の回転、左右反転、上下反転、表示リセットのボタンを追加

- README.md の画像処理エンジン説明に、アニメ/イラスト向けにはReal-CUGANを優先して試す目安を追記
- README.md にGitHub Releasesからのダウンロード案内、上書きアップデート方法、Python公式ダウンロードリンクを追加
- README.md の `run_raiv.vbs` 説明を、VBScript廃止予定を踏まえた注意書きへ修正

## v0.1.4 - 2026-06-02

### 追加 / Added

- その他タブに現在のバージョン表示と手動アップデート確認機能を追加
- GitHub Releases の最新版を確認し、新バージョンがある場合はダウンロード先URLを表示
- ComfyUI API連携によるAI彩色タブを追加
- API形式のworkflow JSONを読み込み、現在画像をComfyUIへ送って彩色結果を保存・表示する機能を追加
- ComfyUI初期設定、推奨モデル確認、RAIV標準workflow自動生成、推奨モデルダウンロード機能を追加
- RAIV標準workflowをSDXL Lineart ControlNet対応に変更
- 推奨ControlNet `shermang_controlnet_standard_lineart_sdxl.safetensors` の検出とダウンロード導線を追加
- ComfyUIで生成エラーが発生した場合、彩色中のまま待ち続けずエラーを表示するように変更
- `RAIV_colorized` に彩色済み画像があれば表示に使用する機能を追加
- 彩色結果をフォルダに保存するか、一時ファイルとして表示するかを選べるように追加
- AI彩色実行時はComfyUI側の恒久出力を避け、RAIV側で保存先または一時ファイルを管理するように変更
- 彩色結果を表示に使うかどうかのオン/オフを追加
- AI彩色の先行処理オン/オフと先読み枚数設定を追加
- AI彩色の先行処理では彩色済みページをスキップし、手動彩色時のみ既存結果を上書きするように変更
- AI彩色の彩色強度、線画保持、彩度補正、輝度保持、Positive/Negative promptをUIから調整できるように追加
- AI彩色の標準サンプラーを `dpmpp_2m_sde` / `karras` / 28 steps に変更

## v0.1.0 - 2026-05-27

初回公開版です。

### 追加 / Added

- Real-CUGAN / Real-ESRGAN による画像の超解像・復元表示に対応
- 画像ファイル、フォルダ、ZIP/CBZ、RAR/CBR、7z/CB7 のドラッグアンドドロップ表示に対応
- 高速なページ送り、ビューアー先読み、エンジン先行処理に対応
- 元画像と処理済み画像を比較する比較モードを追加
- ズーム、パン、回転、反転、全画面表示に対応
- サムネイル/フィルムストリップ表示を追加
- キーボード/マウスのキーコンフィグを追加
- 日本語 / English の UI 言語切り替えを追加
- CPU リサンプルキャッシュと表示リサンプル方式の選択を追加

### 改善 / Improved

- PySide6 + OpenGL ベースの表示により、ページ送りとパン操作の応答性を向上
- 右ペインとサムネイル列の固定表示・自動表示を切り替え可能に
- 設定をアプリ横の `setting.json` に保存する方式に整理
